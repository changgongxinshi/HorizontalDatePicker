# HorizontalDatePicker
android 自定义横向年月日期选择控件


1.为了避免重复造轮子，自定义横向日期选择控件，可以获取从当前月开始，至当前月的前五个月的年月信息。

2.点击后，有相应的信息返回。

3.暂不支持滚动选择时间
